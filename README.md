# EU Scarf Quality Control Standards Guide

A professional reference guide for scarf manufacturers, sourcing teams, QC inspectors, importers, and brand owners working with the European Union market.

This repository provides a structured overview of the most relevant quality control standards, compliance requirements, labeling rules, chemical limits, color fastness benchmarks, and physical performance tests for scarves, shawls, wraps, and neck warmers exported to the EU.

---

## What This Guide Covers

- Chemical and physical safety standards
- Color fastness requirements
- Physical performance benchmarks
- GPSR safety requirements
- Labeling and traceability requirements
- REACH restricted substances quick reference
- OEKO-TEX product class comparison
- Common quality defects and rejection criteria
- Recommended testing frequency
- Standard index and key term glossary
