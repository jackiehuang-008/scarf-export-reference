# HS Codes Import Tariff Reference

A practical reference guide for importers, sourcing teams, compliance reviewers, QC staff, and textile product developers working with scarf classification and tariff review across major markets.

This repository provides a structured overview of HS code logic, fiber-based classification, construction-based distinctions, and indicative import tariff considerations for scarves, shawls, wraps, and related textile accessories.

## What This Guide Covers

- HS code classification logic
- Woven vs. knit product distinctions
- Fiber-based code selection
- Import tariff reference by destination market
- Origin-related tariff considerations
- Documentation requirements at entry
- Common classification mistakes
- Compliance and customs review notes
- Practical reference points for sourcing teams and importers

## Intended Users

- Importers
- Sourcing teams
- Compliance reviewers
- QC teams
- Customs support staff
- Textile product developers
- Private label buyers
- Trade research teams

## Recommended Use Cases

- A sourcing reference
- A customs classification checklist
- A tariff review reference
- A compliance briefing for suppliers
- A pre-shipment documentation reference
- A technical knowledge page on GitHub Pages
